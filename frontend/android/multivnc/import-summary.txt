Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app/src/main/AndroidManifest.xml
* assets/ => app/src/main/assets/
* libs/GoogleAdMobAdsSdk-6.3.1.jar => app/libs/GoogleAdMobAdsSdk-6.3.1.jar
* libs/com.antlersoft.android.contentxml.jar => app/libs/com.antlersoft.android.contentxml.jar
* libs/com.antlersoft.android.db.jar => app/libs/com.antlersoft.android.db.jar
* libs/jmdns.jar => app/libs/jmdns.jar
* lint.xml => app/lint.xml
* res/ => app/src/main/res/
* src/ => app/src/main/java/
* src/.DS_Store => app/src/main/resources/.DS_Store
